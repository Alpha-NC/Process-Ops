
## Prévisualisation

```bash
python -m http.server 8080
```

## Export PDF

1. Ouvrir dans **Chrome**
2. `Ctrl + P` → Enregistrer en PDF
3. Options :
   - Mise en page : **Paysage**
   - Taille : **A4**
   - Marges : **Aucune**
   - ✅ **Graphiques d'arrière-plan**
